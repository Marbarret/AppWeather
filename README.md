# Aplication Weather #

Project made in UIKit, using URLSession for API consumption.

## Tech Requirements ###
* Target deployment iOS 14.4
* Xcode Version 12.4
* Swift 5

## Project Architecture ###
* This project uses MVC as architecture

## Use in Project ###
* Request API - URLSession
* CoreLocation
* Delegate/Protocolos
* @IBInspectable
* Autolayout
* XIB
